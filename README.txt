Controls:
W - move character up, scroll inventory (in inventory mode)
S - move character down, scroll inventory (in inventory mode)
A - move character left
D - move character right
RMB - shoot
U, Enter - use item (in inventory mode)
D, backspace - drop item (in inventory mode)
G - pick up item
I - use inventory
